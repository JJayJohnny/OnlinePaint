# OnlinePaint
Simple paint-like program with online, client-server communication.<br/>
Written entirely in C#. Client app uses a WPF window while server is pure console. It's <b>the best</b> paint experience that you can possibly get. <br/>Tell your friends to download it, so you can all have fun playing with this magnificent program.
